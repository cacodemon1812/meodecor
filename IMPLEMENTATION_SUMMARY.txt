## 🚀 How to Use

### View the Gallery
```bash
npm run dev
# Gallery is live with all enhancements!
```

### Customize Colors
Edit file: `src/styles/tokens.ts`
```typescript
primary: { main: "#YourBrandColor" }
```

### Add More Images
Edit file: `src/components/GalleryGrid.tsx`
```typescript
galleryItems.push({
  id: "new-id",
  src: "/assets/img/path/image.jpg",
  category: "boys",
  alt: "Description",
  title: "Display Title",
  theme: "Theme Name"
})
```

### Adjust Animations
Edit file: `src/styles/gallery.css`
```css
.fade-in {
  animation: fadeInUp 0.8s ease-out forwards; /* Change timing */
}
```
